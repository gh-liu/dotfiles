> https://www.moolenaar.net/habits.html

> https://www.moolenaar.net/habits_2007.pdf

> https://www.youtube.com/watch?v=p6K4iIMlouI

1. 发现效率低的地方: 找到浪费时间的点
2. 寻找更高效率的方法：在线帮助文档、搜索引擎
3. 将第2中寻找到的方法变成习惯：不断提高
