mini-test-skeleton/
├── scripts/
│   └── minimal_init.lua   -- 测试用最小配置（加载 mini.test + 插件）
└── tests/
    ├── test_smoke.lua     -- 冒烟测试（加载/setup/命令存在）
    └── test_actions.lua   -- 功能测试（具体 action 的行为）

使用方法：
1. 把这两个目录（scripts/ 和 tests/）复制到你的插件仓库根目录
2. 修改 scripts/minimal_init.lua 中的 mini.nvim 路径
3. 修改测试文件中的 'myplugin' 为你的插件名
4. 运行测试：
   - 交互式：nvim -u scripts/minimal_init.lua -c "lua MiniTest.run()"
   - Headless：nvim --headless -u scripts/minimal_init.lua -c "lua MiniTest.run()"

CI 示例（GitHub Actions）：
```yaml
- name: Run tests
  run: |
    nvim --headless -u scripts/minimal_init.lua \
      -c "lua require('mini.test').run({ execute = { reporter = require('mini.test').gen_reporter.stdout() } })"
```
